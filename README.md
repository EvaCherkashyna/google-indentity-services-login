## ⚡️ Quick start

#### Node version `v14.18.1`

1. Clone the repo:
```sh
 git clone https://github.com/EvaCherkashyna/Meeting-Room-Incora.git
```

2. Install NPM packages:

```sh
 npm install
```

3.  Run the application:

```sh
 npm run start
```
